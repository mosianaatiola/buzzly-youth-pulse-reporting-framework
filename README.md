# Buzzly Youth Pulse Reporting Framework

## Overview

This repository showcases the reporting framework I developed during my **Data Analytics Internship with Buzzly (Auckland Council)**.

The project focused on developing a reusable reporting framework to streamline data preparation, qualitative analysis and insight reporting for youth engagement initiatives.

> **Note:** This repository is a portfolio showcase only. It does not include confidential datasets, internal workbooks, proprietary code or Buzzly reports.

---

## My Contributions

During my internship, I:

* Developed automated Power Query workflows to import, clean and transform raw data
* Built reusable Excel reporting templates to support consistent reporting
* Designed qualitative coding frameworks for analysing open-text responses
* Conducted thematic analysis to identify key themes and insights
* Produced two professional insight reports:

  * Education & Learning
  * Youth Voice & Participation
* Authored documentation and handover guides to support future analysts
* Applied AI-assisted workflows responsibly to improve reporting efficiency while validating all outputs
